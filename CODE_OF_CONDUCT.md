# Contributor Covenant Code of Conduct (short)

Our community is dedicated to providing an open and welcoming environment for all contributors. We expect everyone to follow these basic guidelines:
- Be respectful and considerate in all interactions.
- Use inclusive language; avoid harassment and derogatory comments.
- Be open to feedback and assume good intent.

If you experience or witness unacceptable behavior, please open an issue or contact the repository owner directly.
